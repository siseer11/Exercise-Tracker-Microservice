Exercise Tracker

1. Can create a new user at : /api/exercise/new-user , by passing in the req body a username.
2. Get all the users at : /exercise/users.
3. Can create a new exercise at : /api/exercise/add. Passing a valid userId, description, duration, and date which is optionally. If not present , it will be automaticly set to today.
4. Retrieve exercises for a user at : /api/exercise/log?userId=yourId. Passing a valid userId, it will return all the exercises for this specific user. In adition you can filter the results by passing aditional parametters. Suported one are : from (date),to (date) and limit(number).
